# Repos on disk

This file records where this workspace and any sibling repos live on your computer, and where each one points online (its git remote). It is a map so anyone (including a future you) can find the code and know which remote it syncs with.

Fill in one row per repo. Use the local path on your machine and the remote URL the repo pushes to.

## When to update

- New repo cloned or created locally: add a row.
- A repo moved on disk: update its local path.
- A remote changed: update the git remote.

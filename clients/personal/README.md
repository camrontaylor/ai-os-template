# Personal Brand (blank template)

This folder is for **your personal brand** - you as an individual. Your personal voice, audience, and content (a personal site, LinkedIn, YouTube, a newsletter) live here, kept separate from any company you run and from the clients you serve.

Why a separate folder: your personal voice is usually different from your company's voice. Keeping them apart means each stays consistent.

## How to set it up

Open this folder and onboarding runs automatically:

```bash
cd clients/personal
claude
```

Claude sees the empty `brand_context/` and runs `/start-here` to build your personal voice, positioning, and audience. Or fill the `brand_context/` templates in by hand.

If you do not have a separate personal brand, delete this folder.

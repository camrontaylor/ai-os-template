# _threads - parked and in-progress work

One card per thing you want to come back to later but not right now. The underscore keeps this folder at the top of `projects/`.

Each card stays short:
- What it is
- What it is waiting on
- The next action
- Where the full work lives

When a card is added or changed here, it should also appear as a to-do on your Notion "Today" task list (dashboard). That sync is wired by `scripts/thread-to-notion.py` (to be built: needs your Notion task database id + NOTION_API_KEY).

This is separate from the per-session `### Open threads` lines in `context/memory/{date}.md`. Those are small loose ends inside one session. This folder is for bigger parked work you will return to.

# Live (skills-library stage 4)

Empty by design. Skills do NOT live in this folder long-term. This is just the final pre-move test space.

After a skill passes `review/`, it should be moved into the AI-OS-wide skills directory at `.claude/skills/<category>-<name>/` and registered in `AGENTS.md`. That is the real "live" location, where AI-OS picks the skill up automatically by trigger word.

So if this folder ever has anything in it for more than a day, it means a promotion was started but not finished.
